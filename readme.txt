为了便于记账，有一个小工具提供给大家，点击这个网址进入：
http://tensor.applinzi.com/newBookkeeping?shopid=
第一次访问这个网页的时候需要输入安全口令：
进去之后是一个网页：[pic1]
默认的日期是当天，这个日期也可以修改为任意一天，只需要点击日期框，然后选择要记账的日期。也可以点击“上一天”、“下一天”来切换。
这个页面分为上下两个部分：上面是支出项，下面是收入项。
支出项分为一个列表和输入区域，在输入区域输入的内容点击“添加”按钮后会在列表中增加一条纪录。
收入项的输入方法和上面支出项类似。
支出和收入都输入完成后（一定要点击“添加”将输入的项目都添加到列表中才生效),最后点击页面最下方的“保存”，把这个日期所有的输入保存起来。（一定要点击“保存”！）

保存某个日期的输入后可以切换到其他日期输入。

在“报帐群"微信群中，如果想知道某天的记账情况，可以在群中发信息：mmj20180101 其中20180101替换为要查询的8位年月日（4位年份，2位月份，2位日期,如想知道今天就输入mmj20180324,注意3月份前面的0不要省略)
我会马上回复那天的记账情况。（It is a magic,you can try!)
